WELCOME TO RECYCLEAI (ENTER jawaadmerali.github.io TO SEE THE PROJECT)

I saw the need for an app that would allow individuals to quickly scan an item and, through analyzing material properties, would be able to identify if the item is recyclable or not. 
In addition, it would provide a message to the user with fun facts about recycling
This primarily relies on OpenAI model and Google Teachable Machine which has 1000+ data points to make an accurate prediction
